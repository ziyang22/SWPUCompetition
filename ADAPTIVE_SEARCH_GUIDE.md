# 自适应搜索模式使用指南

## 概述

自适应搜索模式是一种智能优化算法，通过动态调整计算步长来大幅提升井眼通过能力计算的性能。

## 性能对比

### 测试结果

| 测试范围 | 模式 | 窗口数 | 耗时 | 性能提升 |
|---------|------|--------|------|---------|
| 3300-3400m (100m) | 固定步长 | 200 | 1.90s | - |
| 3300-3400m (100m) | 自适应搜索 | 14 | 0.11s | **17.3倍** |
| 3000-4000m (1000m) | 固定步长 | 2000 | 9.28s | - |
| 3000-4000m (1000m) | 自适应搜索 | 104 | 0.44s | **21.1倍** |

### 关键优势

- ✅ 窗口数减少 93-95%
- ✅ 计算速度提升 17-21 倍
- ✅ 范围越大，性能提升越明显
- ✅ 自动检测卡点并精确定位

## 使用方法

### 命令行格式

```bash
./projection_method <length> <radius> <begin> <end> <step> <enable_adaptive> [growth_factor] [min_step] [max_step]
```

### 参数说明

| 参数 | 说明 | 默认值 | 推荐值 |
|-----|------|--------|--------|
| `length` | 工具长度 (m) | - | 1.0 |
| `radius` | 工具半径 (m) | - | 0.025 |
| `begin` | 起始深度 (m) | - | 3000 |
| `end` | 截止深度 (m) | - | 4000 |
| `step` | 初始步长 (m) | - | 0.5 |
| `enable_adaptive` | 启用自适应搜索 (0/1) | 0 | 1 |
| `growth_factor` | 步长增长系数 | 2.0 | 2.0 |
| `min_step` | 最小步长 (m) | 0.5 | 0.5 |
| `max_step` | 最大步长 (m) | 10.0 | 10.0 |

### 使用示例

#### 1. 固定步长模式（传统方式）

```bash
./projection_method 1.0 0.025 3000 4000 0.5 0
```

#### 2. 自适应搜索模式（推荐）

```bash
./projection_method 1.0 0.025 3000 4000 0.5 1 2.0 0.5 10.0
```

#### 3. 自适应搜索模式（使用默认参数）

```bash
./projection_method 1.0 0.025 3000 4000 0.5 1
```

## 工作原理

### 算法流程

1. **初始阶段**：从起始深度开始，使用初始步长（如 0.5m）
2. **快速跳跃**：如果工具可以通过，步长按增长系数（如 2.0）增长
3. **步长限制**：步长增长到最大值（如 10m）后保持不变
4. **卡点检测**：如果检测到工具无法通过
5. **智能回溯**：回到上一个通过点，使用最小步长精确搜索
6. **精确定位**：在卡点附近使用小步长确保精度

## 参数调优建议

### 增长系数 (growth_factor)

- **保守**: 1.5 - 更多迭代，更精确
- **推荐**: 2.0 - 平衡性能和精度
- **激进**: 3.0 - 更少迭代，可能错过细节

### 最大步长 (max_step)

- **小范围** (< 500m): 5.0m
- **中等范围** (500-2000m): 10.0m
- **大范围** (> 2000m): 20.0m

### 最小步长 (min_step)

- **高精度要求**: 0.1m
- **标准精度**: 0.5m
- **快速扫描**: 1.0m

## 适用场景

### ✅ 推荐使用自适应搜索

- 大范围深度计算（> 500m）
- 井眼相对平滑的区域
- 需要快速获得结果
- 批量计算多个工具尺寸

### ⚠️ 谨慎使用

- 已知存在多个卡点的复杂井眼
- 需要完整的深度-直径曲线
- 极高精度要求的场景

### ❌ 不推荐使用

- 小范围计算（< 50m）
- 研究井眼几何特征变化

## 常见问题

### Q: 自适应搜索会影响精度吗？

A: 不会。在检测到卡点时，算法会自动回溯并使用最小步长进行精确搜索，确保卡点定位精度。

### Q: 为什么有时性能提升不明显？

A: 如果井眼存在多个卡点或变化剧烈，算法会频繁回溯，性能提升会减少。这种情况下可以考虑使用固定步长模式。

### Q: 可以在交互模式下使用自适应搜索吗？

A: 目前自适应搜索只支持命令行模式。交互模式默认使用固定步长。

### Q: 如何选择合适的参数？

A: 对于大多数场景，使用默认参数即可（增长系数 2.0，最大步长 10m）。如果需要更高精度，可以减小最大步长；如果追求极致性能，可以增大最大步长。
